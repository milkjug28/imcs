CUSTOM EYES - FILE GUIDE
=========================

To replace the CSS eyes with your own drawings, create these files:

LAYER 1 - THE EYE SOCKET/WHITE (stays still)
--------------------------------------------
- eye-white-left.png  (or .gif/.svg)
- eye-white-right.png (or .gif/.svg)

This is the background layer - draw the full eyeball shape, 
white part, any eyelids, lashes, bloodshot veins, etc.
Recommended size: 150x150 pixels (or bigger, will be scaled)

LAYER 2 - THE IRIS/PUPIL (moves to follow cursor)
-------------------------------------------------
- eye-iris-left.png  (or .gif/.svg)
- eye-iris-right.png (or .gif/.svg)

Draw JUST the colored iris and black pupil.
USE TRANSPARENT BACKGROUND so only the iris shows.
Recommended size: 60x60 pixels

TIPS FOR MS PAINT VIBES:
------------------------
- Use chunky black outlines
- Don't anti-alias (keep it crispy)
- Weird bloodshot veins = good
- Different colored irises = chaotic and fun
- Can make them asymmetrical for extra cursed look

Once you have your files, let me know and I'll update the HTML!
